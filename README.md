# Recipe Finder

A simple application that suggests recipes based on user-provided ingredients using the Spoonacular API.

## Features
- Search for recipes by ingredients.
- User-friendly interface.

## Technologies Used
- Python
- Tkinter (for GUI)
- Requests (for API calls)
